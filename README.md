## Installation

### npm

```bash
$ npm install morsee --save
```

### yarn

```bash
$ yarn add morsee
```

## Usage

```js
const { encode, decode } = require('morsee');

const encoded = encode('hello world');
// .... . .-.. .-.. ---     .-- --- .-. .-.. -..

const decoded = decode('.... . .-.. .-.. --- / .-- --- .-. .-.. -..');
// hello world
```
